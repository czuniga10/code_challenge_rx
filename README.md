# RedX Code Challenge

### Node.js => PostgreSQL Backend 
#### Backend Installation Instructions
+ In your terminal, cd into code_challenge_rx
+ run ``yarn install`` (installs necessary dependencies) <br>
+ run ``nodemon`` OR ``node server/index.js``
